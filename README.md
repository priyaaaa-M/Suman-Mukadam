# Suman Mukadam - Makeup Artist

A luxury, premium makeup artist website built with Next.js and React.

## Features

- **Next.js App Router**: Modern and fast architecture.
- **Premium Design**: Ivory and gold aesthetic with soft shadows, glassmorphism, and elegant typography (Playfair Display, Great Vibes, Inter).
- **Responsive Layout**: Designed mobile-first for all screen sizes.
- **Components**: Reusable, modular UI components utilizing CSS Modules.

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Framework: Next.js (React)
- Styling: Tailwind CSS & CSS Modules
- Icons: Lucide React
- Fonts: Google Fonts

## Deployment

This application is ready to be deployed to platforms like Vercel, Netlify, or any standard Node.js hosting.

```bash
npm run build
npm start
```
